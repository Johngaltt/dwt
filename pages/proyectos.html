Nuestros Proyectos

Estas son fotos ilusitrativas de algunos de nuestros proyectos.

Software de venta de entradas

El software consta de dos partes: la parte de gestión, en la que se lleva el control general de las visitas, jornadas de los trabajadores o cambios que haya podido haber, y por el otro, el TPV o taquilla en la que los trabajadores pueden administrar las visitas, realizar impresiones de las entradas y organizar los eventos de ese día.

Aplicación móvil

El proyecto consta de dos partes: un sistema de recogida de datos mediante una encuesta y un software para los administradores en el que poder modificar las preguntas, visualizar los datos y tratar la información que ha sido recogida. Este sistema permite la captura de datos no solo en el propio centro sino en cualquier evento o feria de captación de socios, agilizando al máximo el trámite de altas en las instalaciones.

Servicios de Prevención y Seguridad

Lo primero que se hizo fue un estudio pormenorizado de las necesidades a cubrir para la gestión y control de intervenciones así como mejoras posibles a incorporar al proceso de gestión de intervenciones en cliente final. Una vez finalizado el análisis y habiéndolo contrastado con nuestro cliente, se desarrollo de una solución que sirve de herramienta para gestionar de forma ágil las acciones llevadas a cabo.

Control de acceso y presencia

La realización del proyecto consistió en la adaptación de un nuevo sistema de control de acceso con tarjetas de proximidad RFID a dos parking privados, a los que los vehículos acceden a través de tres portillas motorizadas ya existentes. El antiguo sistema de llave magnética, compuesto por seis cerraduras, fue sustuido por lectores de tarjeta de proximidad para regular el control de acceso de entrada y salida por cada una de las portillas, aprovechándose para ello el 100% de la infraestructura existente, incluyendo nuestros lectores de control y electrónicas.

TIENDA ONLINE

El cliente necesitaba una tienda online en la que poder vender los productos de su tienda física y que permitiese una sincronización de stock entre esta y la tienda online. Para la comodidad del cliente, se decidió trabajar con Prestashop. Esta herramienta permite crear una web personalizada a partir de la plantilla que el cliente elija, lo que consigue un resultado atractivo y totalmente personalizado por el usuario. La tienda decidió hacer un diseño fácil e intuitivo para la comodidad de los usuarios, con el fin de que cualquiera pudiera navegar en ella.

Auditoría de posicionamiento SEO

El cliente solicitó una auditoría de posicionamiento a la vez que la creación de su logo y su página web. En este caso, la empresa quería saber cómo tener un buen posicionamiento SEO para poder darse a conocer de manera online y atraer a más usuarios.Cuando el cliente solicitó este servicio no contaba todavía con la página web. Por ello, se decidió que a medida que se creaba la web se iba a tener en cuenta el posicionamiento que se quería lograr para aplicar todas las directivas necesarias y cumplir con este objetivo.

DISEÑO WEB para pyme

La página web cuenta con un diseño responsive, significando esto que la visualización de la página se adapta a cualquier dispositivo que el usuario emplee. Esto proporciona al usuario una experiencia de navegación óptima. Cabe destacar que, a lo largo del proceso de creación, la coordinación y colaboración con el cliente, quien ha facilitado las imágenes y los textos a publicar, ha sido clave para generar una web muy personalizada.

Software para Restaurante

La web está dividida en diferentes secciones. Por un lado, el cliente quería exponer información sobre el chef y los productos, como se mencionaba anteriormente. Por otro, el cliente vio la importancia de crear una sección para los comentarios en redes sociales y sus enlaces directos, con el fin de que cualquier usuario pudiera comenzar a seguir las cuentas o compartirlas.

Sistema de registro y gestión

El desarrollo software se ha creado mediante MICROSOFT .Net que engloba un conjunto de tecnologías destinadas al desarrollo de aplicaciones para el sistema operativo Windows, en este caso se desplegó el proyecto en un entorno local con conexión a los servidores centrales situados en otra comunidad para el alojamiento de los datos.
